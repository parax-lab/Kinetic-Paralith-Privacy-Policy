
# Kinetic Paralith Privacy Policy

Kinetic Paralith does not collect, store, or share any personal user data.

## Data Collection
We do not collect any personal information from users.

## Ads and Tracking
This app does not use:
- Advertisements
- Third-party trackers
- Analytics tools

## User Accounts
No account or sign-in is required to use the app.

## Payments
If you choose to support the developer through the optional "Buy Me a Coffee" feature, the transaction is handled securely by third-party payment providers. The app does not store any financial information.

Kinetic Paralith is safe for all ages and does not knowingly collect any data from children.
